Que es flexbox?

Herramienta que con unas cuantas propiedades, se va a poder realizar diseños flexibles y complejos con menor cantidad de codigo

Ademas sirve para linear los elementos de manera horizontal y vertical

Flexbox es un conjunto de propiedades, y responsive web design es una tecnica de diseño

Reglas de oro para asimilar y entender flexbox

  • Flex necesita un padre y por lo menos un hijo
  • El Flex-container tiene dos ejes: cross-axis (vertical), y Main axis (horizontal)
  • Podemos modifiacr el eje principal con la propiedad: Flex-direction
  • podemos permitir el salto de columnas con Flex-wrap

    Cuando el contenido es mas grande que el contenido, se usa Flex-wrap:nowrap;

    Si queremos que el contenedor se agrande a medida que el contenido aumenta tambien se usa, flex-wrap:wrap;

  • alineamos los elementos en el eje principal con Justify-content
  • alineamos los elementos en el eje secundario con align-items
  • para alinear los hijos de forma individual se usa align-self
  • los hijos flexibles ignoran las propiedades float, clear y vertical align
  • podemos cambiar el tamaño de los hijos con Flex-grow, flex-shrink, y flex-basis

    • por defecto el tamaño de los hijos se define por el tamaño de su contenido

    • Flex-Grow:define el tamaño que crecera un hijo, en relacion a sus demas hermanos, cuando hay un espacio disponible en el contenedor. Su valor por defecto es "0". este trabaja si el contenedor tiene espacio libre, asi, poniendo esta propiedad, se divide entre lso que tienen esta propiedad
    • Flex-shrink: define el tamaño de reduccion de un hijo en relaicon a sus demas hermanos, cuando falte espacio en el contenedor. el valor por defecto es "1"
    • Flex-basis: define el tamaño inicial del hijo. sy valorpor defecto es auto. puede suplantar a la propiedad height y width
  • todo se puede resumir on la propiedad FLEX: (flex-grow, flex-shrink, flex-basis
  • se puede reordenar los hijos flexibles con la propiedad ORDER
